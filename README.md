# Customer Management System

The Customer Management System is a desktop Application in Java that helps manage customers, materials,
quoation, projectes, and inventory in an organized and automated way.

The system allows users to create customer records, manage material, generate quoatation, create new 
projects, and automatically calculate totat payments. It also updates inventory level dynamically when 
materials are used in projects.

The system is designed to improve efficiency, reduce manual errors, and provide better busniess insights
through statistics and automated calculation.

Features:
. CRUD operations for customer, material and project details
. Inventory control system
. Automatic gross payment calculation
. Generate quoatation for the customers
. Search functionality for:
    Customers
    Materials
    Projects
    Inventory
. View system statistcs and income bar graph


Technologies used:
Java
JavaFX
H2 database
IntelliJ IDEA
Git & Github

System Workflow:
1. Add customers
2. Add materials
3. Create new project
4. Select materials for the project
5. Quotation for the customer
6. View inventory
7. Use the search feature to quickly find customers, materials, projects or inventory
8. View statistics and bar graph

How to run:
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run the Main.java file

Project Purpose:
. Object-Oriented Programming (OOP)
. Business logic implementation
. Inventory management system design
. Search filtering functionality
. Automated financial calculation
. Desktop application developement using java
